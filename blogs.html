Scopes in JavaScript

The scope is a policy that manages the accessibility of variables. The accessibility of variables is limited by the scope where they’re created. You are free to access the variable defined within its scope.

What is a Promise in JavaScript ?

A promise is an object that may produce a single value some time in the future: either a resolved value, or a reason that it’s not resolved (e.g., a network error occurred).
